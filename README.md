# pycat


### Description

pycat is a lightweight Python script that mimics the unix cat command on windows.
It recursively searches the entire drive for the specified file inputted (starting from 'C:/')

### Built with
```
Python *3.13.3*
```


### Getting Started  
**Usage:**

```bash
python3 pycat.py
```
